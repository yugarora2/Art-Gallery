About Us

Yug Arora

Hello! I am Yug Arora, the creator of this Indian Art & Culture Gallery website. This project is designed to explore the rich and diverse art forms of India through an interactive map interface.

I worked on designing the layout using HTML and CSS, and added interactivity using JavaScript such as clickable maps, navigation, and user input forms.

My goal for this project was to create a visually engaging and user-friendly platform that helps users discover art from different regions of India.
